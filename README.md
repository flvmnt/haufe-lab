# Stack my Overflow

A Q&A web app where developers can ask programming questions, post answers, and vote on what's helpful — with an AI assistant that drafts answer suggestions when no human has replied yet.

## What it does

- Post questions with code snippets and tags
- Browse and search the question feed
- Answer other people's questions and accept the one that solved your problem
- Upvote and downvote questions and answers so good content rises
- User accounts with a simple reputation score that goes up when your contributions get upvoted
- AI-generated answer suggestions powered by an LLM, so quiet questions still get a starting point

## Tech stack

- **Frontend:** React
- **Backend:** Node.js with Express
- **Database & auth:** Supabase
- **LLM:** Groq for hosted inference, Ollama for local development

## Status

Built across the labs of the Stack my Overflow course — each lab adds another piece of the app.
