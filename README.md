# Stack my Overflow

A Q&A app like Stack Overflow with an AI that posts a draft answer when nobody else has.

## Features

- Ask questions with code and tags
- Browse and search
- Answer questions, accept the one that worked
- Upvote and downvote
- Accounts with a simple reputation score
- AI draft answers (Groq in production, Ollama locally) when a question has no human reply

## Stack

- React
- Node + Express
- Supabase (db + auth)
- Groq / Ollama for the LLM
